Discord Piece V.02

Program note

Though it heavily limits opportunities for live group improvisation, the COVID-19 pandemic encourages the use of electronic stages, which offer a variety of new avenues of expression. This piece uses Discord's "Voice Channel" system to divide a large jam session into isolate sub-groups, a process not feasible in a physical space. This allows for a variety of unlikely interactions, and creates an array of semi-related musical threads.

Directions for coordinator

Server Setup

Create a Discord server with a number of Voice Channels equal to the number of performers divided by two, rounded down. The first channel on the list must be given the name "MainStage". Create a role in this server titled "performer" and assign this role to all of your performers.

Bot Setup

Requires python 3.9+ and discord.py.

pip install discord

Clone this Github repository on your local machine. Create a Discord bot with administrative privledges at the Discord Developer site. Invite this bot to your server. Copy the OAuth2 token into the space in the main.py file. Finally, simply run the script:

python3 main.py

Running the Performance

Once the bot is configured and running, and all the performers are assigned the "performer" role waiting in the "MainStage" voice channel, type the following command directly in the Discord chat:

.setup

If you receive the message "Setup Complete", the piece is ready to run. Enter the following command and the piece will commence:

.begin

Directions for performers

Setup

Using a computer(mobile will not work), follow the discord server invite sent to you by the coordinator. If you do not have a discord account, set one up. Next, click on "User Settings" and under app settings click on "Voice and Video". Turn off "Echo Cancellation", "Noise Reduction" and "Automatic Gain Control". Finally, join the voice channel "MainStage" by clicking on it. Note-Discord by default does not offer stereo audio. However, if you so desire, using a third party application such as Ripcord you can enable it.

Performance

In this piece, you will be shuffled through various audio channels, meeting a variety of other improvisers. In each new setting you enter, try to set up an atmosphere, and carry traces of this atmosphere you develop in all your interactions throughout the duration of the piece.
